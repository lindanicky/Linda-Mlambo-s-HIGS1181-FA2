# Linda Mlambo's HIGS1181 FA2
Top-Down Shooter (FA2)
This project is a 2D Top-Down shooter game developed in unity 6as part of the HIGS1181 Formative Assessment 2

 Features
-player moves using W, A, S, D keys
-sprint using the shift key
-shootong system with bullets
- enemies placed randomly on a 12 by 12 grid
- enemy counter UI
- collision detection between player, bullets and enemies
- win condition when all enemies are destroyed

  Scripts included
  - PlayerController: Controls movement, sprinting and shooting
  - GameManager: Manages enemy count, UI updates and win condition
  - GridManager: Generates grid and spawn enemies
 
    How to play
  -Move: W, A, S, D keys
  -Sprint: Spacebar or mouse click

  Unity version: Unity 6000.0.63f1 LTS
